# 📱 Dialysis Management System - Mobile App

Flutter mobile application for dialysis patients.

## Current Implementation

### Features (Sign Up & Login Only)
- ✅ Patient registration (Sign Up)
- ✅ Email verification (OTP)
- ✅ Login with email or username
- ✅ Password recovery
- ✅ Secure logout
- ✅ Form validation
- ✅ Error handling

## Tech Stack

- **Framework:** Flutter 3.x
- **Language:** Dart
- **Backend:** Firebase
- **IDE:** Visual Studio Code

## Prerequisites

- Flutter SDK installed
- Android Studio with emulator
- VS Code with Flutter extensions

## Setup Instructions

### 1. Create Flutter Project
```bash
cd mobile-app
flutter create patient_app
